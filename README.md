# Noah-Waisbrod/Autonomous_CO2_Mapping_Rover
## MREN 203 course Design Project
An autonomous rover named the MNM MK1 CO2 Detection Robot with real-time navigation, speed regulation, and CO2 mapping capabilities of the air quality for an indoor environment.
